LeadScaper Platform

Архитектура платформы

Слой 1

Поиск компаний (Discovery Layer)

Найти все возможные компании-клиенты по заданным критериям.

#ИнструментФункцияИсточник данных
1.1 AI_Find_Customer Поиск компаний через AI-агентов по ключевым словам, отрасли, геолокации Google, B2B-платформы
1.2 GitHub Lead Signals Поиск компаний по технологическому стеку (GitHub активности) GitHub API
1.3 Clutch Profile Scraper Сбор компаний из B2B-каталога Clutch.co
1.4 TrendScan Поиск трендовых компаний Crunchbase, LinkedIn, Twitter/X
1.5 ICP Researcher Определение Ideal Customer Profile и поиск по нему Веб-поиск, LinkedIn, Crunchbase

      "relevance_score": 0.92

Слой 2

Сбор контактов и соцсетей (Enrichment Layer)

Для каждой найденной компании собрать контакты, соцсети и публичные данные.

#ИнструментФункцияИсточник данных
2.1 ColdReach-Scraper Скрапинг сайта: контакты, миссия, услуги Сайт компании
2.2 lead-generator Поиск LinkedIn-профилей сотрудников и обогащение email LinkedIn, Apollo.io
2.3 linkedIn_scraper Сбор профилей ЛПР с AI-генерацией «болевых точек» LinkedIn
2.4 The 3rd Eye Сбор цифрового следа из всех соцсетей Twitter, LinkedIn, Instagram, Facebook, Telegram
2.5 Business Researcher Глубокое исследование: финансы, структура, рынок Tavily API, веб-поиск

Слой 3

Анализ и генерация инсайтов (Analysis Layer)

Анализ данных для поиска триггерных слов, ЛПР и генерации персонализированных сообщений.

#ИнструментФункцияИсточник данных
3.1 Automated-Decision-Making-Discovery Выявление ЛПР (CEO, CTO, Head of X) с валидацией контактов Веб-поиск, LinkedIn, AI
3.2 OWASP SocialOSINTAgent Анализ текста и изображений из соцсетей, создание отчётов X, Reddit, GitHub, YouTube
3.3 Obsei Анализ тональности и классификация сообщений X, Reddit, Facebook, отзывы
3.4 Company Research Agent (Tavily) Глубокий анализ и генерация отчётов в PDF/Markdown Tavily API, Anthropic API

Слой 4

Вовлечение в соцсетях (Engagement Layer)

Автоматический мониторинг и вовлечение аудитории по триггерным словам.

#ИнструментПлатформаФункция
4.1 Agent-X (TuitBot) X (Twitter) Мониторинг ключевых слов, AI-ответы на посты с одобрением
4.2 RedSignal Reddit Поиск «тёплых» лидов, AI-фильтрация релевантности
4.3 n8n workflow Reddit, LinkedIn, X, Instagram Мультиплатформенный мониторинг и уведомления
4.4 Obsei X, Reddit, Facebook Анализ тональности упоминаний

Процесс обработки данных

Последовательность шагов от поиска компании до вовлечения.

→Поиск компаний (Слой 1)
→Обогащение данными (Слой 2)
→Анализ и генерация инсайтов (Слой 3)
→Запись в CRM (PostgreSQL / Salesforce)
→Мониторинг соцсетей по триггерным словам (Слой 4)
→AI-фильтрация и классификация (Obsei + OWASP)
→Генерация персонализированных сообщений (Agent-X / RedSignal)
→Вовлечение и отправка в CRM
